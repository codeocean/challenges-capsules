# Results — Challenge 09: BindCrafting

## Evidence Strength: PARTIAL — Pipeline validated with simulated AF2 metrics against real target structure

The pipeline produces ranked binder candidates with realistic structural metrics, but all candidate designs are **computationally simulated** — not generated by actual BindCraft/AlphaFold2 runs. The target protein (PDB 1RK9, Parvalbumin) is real, downloaded from RCSB.

## Why Results Are Simulated

BindCraft and RFdiffusion/ProteinMPNN require:
- **GPU compute** (not available in the standard capsule environment)
- **AlphaFold2 model weights** (~3.5 GB, not pre-installed)
- **BindCraft software** (specialized installation)

The pipeline demonstrates the full **post-design analysis workflow**: filtering, ranking, fusion compatibility assessment, and wet-lab output packaging — but using simulated candidate structures instead of real AF2 predictions.
